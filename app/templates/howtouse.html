How To Use


To generate an image of either a dog, a math symbol, or a Marvel inspired superhero, follow the steps below.
  1. Go to the link (by clicking the options at the top) of the respective image category.
  2. To generate a dog or a math symbol, input a valid dog species or math symbol into the input box (valid symbols/species must be entered as seen below but are not case sensitive).
  3. Once you have gotten to this point, click the generate button (or click the Enter button if you are generating dogs/math symbols) to generate the images.
  4. Images will be generated shortly after, and clicking the generate button again will restart the process of creating images.

* Some important things you should know are:
  1. If you refresh the page once images are generated, it will generate new images of the same species/symbol.
  2. If you want to get back to a page without any images generated, you can click the link in the top for the page you are on.
  3. If you want to see the valid species/symbols once the images are generated, you can scroll down.
